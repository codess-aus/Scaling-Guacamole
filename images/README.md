# Images Directory

Store hero images and other blog assets here.

## Guidelines

- **Format**: Use WebP or JPEG for photos, PNG for graphics with transparency
- **Size**: Keep files under 500KB (compress before uploading)
- **Dimensions**: Hero images should be 1200x600px or similar 2:1 ratio
- **Naming**: Use descriptive names like `hero-capturedecisions.jpg`

## Optimization Tools

- [TinyPNG](https://tinypng.com/) - Compress PNG/JPEG
- [Squoosh](https://squoosh.app/) - Modern image compression
- [ImageOptim](https://imageoptim.com/) - Mac app for optimization

## Usage in HTML

```html
<img src="../images/hero-capturedecisions.jpg" alt="Description">
```
